## Quick Links

### For First-Time Users

1. Start with [Getting Started](./getting-started.md)
2. Run `pip install dockai-cli`
3. Set your API key in `.env`
4. Run `dockai build .`

### For DevOps Engineers

1. Review [Configuration](./configuration.md) for all options
2. Set up [GitHub Actions](./github-actions.md) for CI/CD
3. Create `.dockai` files for repo-specific behavior

### For Platform Teams

1. Read [Architecture](./architecture.md) to understand the workflow
2. Follow [Customization](./customization.md) for organizational standards
3. Use [GitHub Actions](./github-actions.md) for CI/CD integration

### For AI Agents

1. Read [MCP Server](./mcp-server.md) for integration details
2. Configure your MCP client (e.g., Claude Desktop)
3. Use `dockai` tools directly in your chat
